#### 优先级排序（艾森豪威尔矩阵）

```
四象限分类：

紧急 + 重要 → 立即执行（今天必须做）
重要 + 不紧急 → 计划执行（安排时间做）
紧急 + 不重要 → 委托他人（可以请别人做）
不紧急 + 不重要 → 删除（考虑是否真的需要）
```

### 4. 文件管理

#### 批量重命名

```bash
# 按日期前缀重命名当前目录图片
for f in *.jpg *.png; do
  DATE=$(date -r "$f" +%Y%m%d)
  mv "$f" "${DATE}_${f}"
done

# 统一转小写
for f in *; do
  mv "$f" "$(echo $f | tr '[:upper:]' '[:lower:]')"
done

# 添加序号前缀
i=1; for f in *.pdf; do
  mv "$f" "$(printf '%03d' $i)_${f}"; ((i++))
done

# 批量替换文件名中的空格为下划线
for f in *\ *; do mv "$f" "${f// /_}"; done
```

#### 文件整理脚本

```bash
#!/bin/bash
# organize_files.sh - 按类型整理文件到对应目录

SRC_DIR="${1:-.}"

declare -A TYPE_MAP=(
  ["images"]="jpg jpeg png gif webp svg bmp"
  ["documents"]="pdf doc docx xls xlsx ppt pptx txt md"
  ["videos"]="mp4 mov avi mkv wmv"
  ["audio"]="mp3 wav flac aac ogg"
  ["archives"]="zip tar gz 7z rar"
  ["code"]="py js ts jsx tsx html css json yaml yml sh"
)

for category in "${!TYPE_MAP[@]}"; do
  mkdir -p "$SRC_DIR/$category"
  for ext in ${TYPE_MAP[$category]}; do
    find "$SRC_DIR" -maxdepth 1 -name "*.$ext" -exec mv {} "$SRC_DIR/$category/" \;
  done
done

echo "文件整理完成"
```

#### 文件搜索与统计

```bash
# 按大小找出占用空间最大的文件（Top 10）
find . -type f | xargs du -h | sort -rh | head -10

# 统计各类型文件数量
find . -type f | sed 's/.*\.//' | sort | uniq -c | sort -rn

# 查找最近 7 天修改的文件
find . -type f -mtime -7 -not -path '*/\.*'

# 查找重复文件（基于 MD5）
find . -type f | xargs md5sum | sort | awk 'BEGIN{prev=""} $1==prev{print $2} {prev=$1}'
```

### 5. 时间规划

#### 每日时间块规划

```
时间块建议（根据个人精力曲线调整）：

上午 9:00-12:00 → 深度工作（攻坚难题、创作、重要决策）
午后 13:00-15:00 → 轻度工作（回复消息、简单处理、会议）
下午 15:00-18:00 → 执行工作（常规任务、文档、代码）
碎片时间 → 收件箱清零、快速回复、阅读积累

关键原则：
- 每天 3 件最重要的事（MIT - Most Important Tasks）
- 番茄工作法：25分钟专注 + 5分钟休息
- 每周五做下周规划，每天早上做当天规划
```

#### 时间追踪脚本

```bash
#!/bin/bash
# time_tracker.sh - 简单时间追踪

LOG_FILE="$HOME/time_log.txt"

start_task() {
  echo "$(date '+%Y-%m-%d %H:%M:%S') START $1" >> "$LOG_FILE"
  echo "开始记录：$1"
}

end_task() {
  echo "$(date '+%Y-%m-%d %H:%M:%S') END $1" >> "$LOG_FILE"
  echo "结束记录：$1"
}

# 用法
# ./time_tracker.sh start "写周报"
# ./time_tracker.sh end "写周报"
case "$1" in
  start) start_task "$2" ;;
  end)   end_task "$2" ;;
  log)   cat "$LOG_FILE" | tail -20 ;;
  *) echo "用法: $0 start|end|log [任务名]" ;;
esac
```

### 6. 快速信息处理

#### 常用文本处理命令

```bash
# 统计文件行数/词数/字符数
wc -lwc file.txt

# 提取 Markdown 中所有标题
grep '^#' document.md

# 提取文本中所有 URL
grep -oE 'https?://[^[:space:]]+' file.txt

# 统计关键词出现频率
tr '[:space:]' '\n' < file.txt | sort | uniq -c | sort -rn | head -20

# 合并多个文本文件并去重
cat *.txt | sort -u > merged.txt

# JSON 格式化
cat data.json | jq '.'

# CSV 快速预览（带格式）
column -t -s',' data.csv | head -20
```

## 工作流程

### 信息整理流程

1. **收集**：汇总所有输入（聊天记录、文档、邮件、笔记）
2. **提炼**：识别关键信息（决策、行动项、截止日期、负责人）
3. **结构化**：套用合适模板组织信息
4. **归档**：按项目/日期分类存档，便于检索

### 日常任务处理流程

1. **早晨规划**（10分钟）：确认今天 3 件最重要的事
2. **工作中**：用时间块专注，减少切换
3. **每日收尾**（15分钟）：完成日报，整理待办
4. **每周回顾**（30分钟）：写周报，规划下周，回顾目标
